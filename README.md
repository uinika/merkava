# Merkava

A seed project with STM32F10x standard peripheral library.

![logo](./assets/logo.jpg)

## Base on

- MCU **STM32F103C8T6**
- Keil uVision V5.28.0.0
- STM32F10x standard peripheral library 3.5.0.

![](./assets/mcu.jpg)

## Directory description

- **CMSIS**：内核驱动程序
- **Lib**：内部功能基本函数库
- **Startup**：汇编编写的启动程序
- **User**：用户主函数以及其它用户程序
- **Basic**：内部功能驱动函数
- **Hardware**：外部硬件驱动函数
