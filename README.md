# Merkava

A seed project with STM32F10x standard peripheral library.

![logo](./assets/logo.jpg)

## Base on

- MCU **STM32F103ZET6**
- Keil uVision V5.28.0.0
- STM32F10x standard peripheral library 3.5.0.

![](./assets/mcu.jpg)

## Directory description

- **CMSIS**：内核驱动程序；
- **library**：基本固件库；
- **startup**：汇编启动程序；
- **sources**：用户功能代码；
- **common**：封装的通用功能；
- **driver**：BSP 板级驱动程序。
